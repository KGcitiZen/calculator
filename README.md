## Hotkeys

* Operands
  * 0-9.
* Operators
  * Plus: +
  * Minus: -
  * Divide: /, :
  * Multiply: *
  * Equal: =, 'Enter'
  * Plus-minus: 'KeyT'
  * Modulo: %
* Functions
  * Clear last: 'Backspace'
  * Clear all: 'KeyC'
